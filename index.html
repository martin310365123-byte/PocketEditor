核心功能

去無聲段

偵測長時間靜音，自動裁切空白片段，讓敘事更緊湊。

字幕自動生成

Whisper STT 轉文字，支援繁中與時間碼修飾。

硬燒與樣式

內建字型、大小、邊距設定，避免跳脫字與亂碼。

隨機 BGM + Ducking

自動挑選音樂與人聲避讓，維持語音清晰度。

一鍵合成

MoviePy + FFmpeg 串接輸出，簡化繁瑣步驟。

可離線運作

在本機 Windows 執行，適合長片與內網環境。

如何使用

1. 準備素材
將影片丟入 input/ 資料夾。
2. 設定參數
依需求調整字幕 CPS、最短/最長顯示等。
3. 執行
點選批次檔或於 CLI 下指令。
4. 輸出成片
完成後會出現在 output/。

常見問題

需要網路嗎？
不需要。PocketEditor 可於 Windows 本機離線執行。
字幕亂碼怎麼辦？
請使用「Microsoft JhengHei UI」或其他含中文字型。
是否支援 GUI？
目前為 CLI，後續會推出 GUI 測試版。
