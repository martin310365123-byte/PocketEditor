AI 自動剪輯，一鍵完成

去無聲段、字幕自動生成與硬燒、隨機 BGM + Ducking、合成輸出。支援中文工作流程，可離線。

Windows / 本機運算 MoviePy + FFmpeg Whisper STT
回報問題 / 提案 
$ pocketeditor run \
  --process-all \
  --stt --stt-model small --stt-device cpu --stt-compute int8 \
  --lang zh \
  --fontname "Microsoft JhengHei UI" --fontsize 28 --marginv 40 \
  --concat --concat-which burnin \
  --music-dir "C:\Users\user\AI Editor\music" \
  --post-bgm-vol 0.12 \
  --subs-cps 18 --subs-min-dur 1.2 --subs-max-dur 5.0 --subs-merge-gap 0.4

核心功能

去無聲段

偵測長時間靜音，自動裁切空白片段，讓敘事更緊湊。

字幕自動生成

Whisper STT 轉文字，支援繁中與時間碼修飾，CPS/最短/最長持續時間可調。

硬燒與樣式

內建字型、大小、邊距設定，避免跳脫字與亂碼。

隨機 BGM + Ducking

自動挑選音樂並在人聲時段降低音量，維持語音清晰。

一鍵合成

MoviePy + FFmpeg 串接輸出，簡化繁瑣步驟。

可離線運作

在本機 Windows 執行，適合長片與內網環境。

如何使用

1. 準備素材
將影片丟入 input/ 資料夾。
2. 設定參數
依需求調整字幕 CPS、最短/最長顯示等。
3. 執行
點選批次檔或於 CLI 下指令。
4. 輸出成片
完成後會出現在 output/。
