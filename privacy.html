PocketEditor

隱私權政策

最後更新：2025-09-19

1. 適用範圍

本政策適用於（a）我們的軟體 PocketEditor，以及（b）本官方網站與其提供之文件頁面（以下合稱「本服務」）。若您透過本服務連結至第三方網站或服務，該第三方之隱私作法不在本政策之內。

2. 我們如何處理資料

  • 本機處理為預設：PocketEditor 於您的電腦本機執行，不需將素材或輸出上傳至遠端伺服器。
  • 不預設蒐集個資：我們不會在您不知情的情況下，自動傳送可識別個人之資料給我們。
  • 選用遙測（未開啟）：目前未啟用遙測或分析。若未來提供匿名使用統計，我們會於軟體或網站清楚告知並徵詢您的同意。

3. 日誌與錯誤報告

  • 本機日誌：軟體可能在您的裝置上產生處理日誌以利除錯；日誌留在本機，除非您主動回報問題並附上。
  • 問題回報：若您於 GitHub Issues 或其他管道回報問題，請避免上傳含個資或機密之素材；我們將依據 GitHub 隱私政策 處理。

4. 網站與第三方服務

  • GitHub Pages：本網站由 GitHub Pages 代管，可能由 GitHub 產生必要的伺服器記錄（如 IP、錯誤記錄）。詳見其隱私政策。
  • YouTube 影片嵌入：我們使用 youtube-nocookie.com 進行嵌入，僅在您播放時可能由 YouTube 設定必要 cookie 或追蹤技術。詳見 Google 隱私政策。
  • 下載：軟體經由 GitHub Releases 提供下載，該流程受 GitHub 的條款與隱私政策拘束。

5. 您的選擇與權利

  • 您可選擇不提供任何資料；多數功能仍可在本機使用。
  • 如您向我們提供聯絡資訊（例如支援信箱），可隨時來信要求查詢、更新或刪除。
  • 如您位於具備個資保護法規之司法轄區（例如 GDPR），您可能享有存取、更正、刪除、限制處理與資料可攜等權利；請來信提出。

6. 資料保存

我們不會在伺服器上保存您的素材或輸出。若您透過支援信箱或議題追蹤提供資訊，我們將在處理目的完成或法定保存期間屆滿後刪除或匿名化。

7. 資安

我們持續改善軟體與網站之資安實務，例如最小權限原則、依賴套件定期更新，以及網站僅讀部署。請定期更新至最新版本。

8. 兒少保護

本服務並非為 13 歲以下兒童設計。若您為未成年人，請於法定代理人監護下使用。

9. 本政策之變更

我們可能更新本政策並於頁面上標示「最後更新」日期。重大變更將以醒目方式告知。

10. 聯絡我們

隱私相關問題，請來信：privacy@pocketeditor.app（可改為您的正式信箱）。
