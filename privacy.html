隱私權政策

最後更新：2025-09-19

感謝您使用 PocketEditor（以下稱「本軟體」）與本官方網站（以下與本軟體合稱「本服務」）。我們尊重並重視您的隱私。本政策說明我們如何在最小化資料蒐集的前提下處理資料，以及您所擁有的權利。

1. 適用範圍

本政策適用於： (a) 本軟體在您裝置上的運作；(b) 本網站（含文件、下載與支援頁）。若您連至第三方網站或服務（例如 YouTube、GitHub），其隱私政策由該第三方訂定。

2. 我們如何處理資料

  • 本機離線處理：本軟體預設於您的電腦本機執行，不會自動上傳素材、輸出或個人資料到我們的伺服器。
  • 不預設蒐集個資：我們不會在您不知情的情況下，主動蒐集可識別個人的資料。
  • 選用遙測（目前未啟用）：若未來提供匿名使用統計或錯誤回傳，將於軟體或網站明確告知並徵得您的同意。

3. 日誌與錯誤報告

  • 本機日誌：為便於除錯，本軟體可能於本機產生處理日誌；日誌不會自動上傳。
  • 問題回報：您可透過 GitHub Issues、電子郵件等方式主動提供資訊。請避免上傳含個資或機密之素材；相關資料將依用途處理後刪除或匿名化。

4. 網站與第三方服務

  • GitHub Pages 託管：本網站由 GitHub Pages 提供。伺服器可能產生必要連線記錄（如 IP、錯誤日誌）。詳見 GitHub 之隱私政策。
  • YouTube 影片嵌入：使用 youtube-nocookie.com 以減少追蹤；您播放影片時，YouTube 可能寫入必要 cookie 或使用追蹤技術。詳見 Google 隱私政策。
  • 下載（Releases）：軟體經由 GitHub Releases 提供，相關互動受 GitHub 隱私政策 與服務條款拘束。
  • Cookie：本站不主動放置行銷追蹤 cookie；第三方服務可能依其政策使用必要 cookie。

5. 您的選擇與權利

  • 您可選擇不提供任何個人資料；多數功能仍可在本機使用。
  • 如您提供聯絡資訊（如電子郵件），可隨時要求查詢、更新或刪除。
  • 若您位於具個資保護法之司法轄區（如 GDPR），您可能享有存取、更正、刪除、限制處理與資料可攜等權利；請來信聯繫。

6. 資料保存

我們不在伺服器保存您的素材或輸出。因支援或客服用途而取得之資料，將在目的完成或法定保存期間屆滿後刪除或匿名化。

7. 資訊安全

我們採行合宜的技術與管理措施維護安全，例如最小權限、依賴套件定期更新、網站僅讀部署。亦請您維持良好資安習慣並更新至最新版本。

8. 兒少保護

本服務非為 13 歲以下兒童設計。若您為未成年人，請在法定代理人監護下使用。

9. 政策變更

重大變更將以醒目方式告知；頁面上方會標示「最後更新」日期。新版本發布後，您持續使用即視為同意變更。

10. 聯絡我們

隱私問題請來信：privacy@pocketeditor.app（可改為您的正式信箱）。
